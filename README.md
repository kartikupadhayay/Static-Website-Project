🖥️ Website Overview
A fully static, single-page corporate website built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no build tools, no dependencies.
Sections

Hero — Company tagline and CTAs
About — Company overview, directors, licence info, embedded company profile PDF
Services — 6 core service categories with tags
Clients — 8 major clients with active project badges
Projects — Tabbed project showcase (Electrical & C&I, Overhauling, Upcoming, Manpower)
Gallery — 5 real site photos with lightbox modal
Tools & Equipment — Owned testing and mechanical equipment list
Why Us — 6 key differentiators
Careers — 3 open job roles + open application CTA
Contact — Enquiry form (Formspree + WhatsApp integration)
Footer — Links, registration details, quick contacts

HTML5:- Page structure and semantic markup
CSS3:- Custom styling, CSS Grid, Flexbox, animations
Vanilla JavaScript:- Scroll reveal, tabs, gallery modal, form handler
Google Fonts:- Montserrat + Open Sans
Formspree:- Contact form email delivery (no backend)
WhatsApp:- APIEnquiry notification via wa.me link

🚀 Deployment
This is a static website — no server-side code required.
Hosted on
GoDaddy Web Hosting Economy — files uploaded to public_html via cPanel File Manager
Domain: sjospl.com (registered on GoDaddy)


To deploy updates
Edit index.html locally
Log in to GoDaddy → My Hosting → Manage → File Manager
Open public_html
Delete old index.html → Upload new one
Changes are live immediately


⚙️ Contact Form Setup (Formspree)
The contact form uses Formspree for email delivery.
To activate:

Sign up at formspree.io
Create a new form → copy your Form ID (e.g. xpwzqabc)
In index.html, find:

   https://formspree.io/f/#####

Replace ##### with your actual ID
Re-upload index.html

On form submission:
✉️ Email sent to registered inbox via Formspree
💬 WhatsApp message opened with pre-filled enquiry details
